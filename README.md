# Medical-Unit-Finder
Medical Unit Finder: aplicativo para a localização E indicação de unidades de atendimento médico
# Linguagem, frameworks e ferramentas utilizadas

- React Native
- React Navigation
- React Native Vector Icons
- React Hooks
- Axios
- Expo
- React-native-maps 
- Goolgle api
- React-native-permissions
